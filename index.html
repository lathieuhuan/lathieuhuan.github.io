My proudest contribution on the front end is a form control system built on top of the Ant Design's form. It adds extra features such as:

  • Tracking the entire form's validity since they want to disable the submit button when the form has any invalid field. No complex configuration is required. You only need to use the Form component and listen to the valid state of the form manager.
  • A FormGroup component wrapping the Ant Design's Collapse. It automatically tracks the validity of the fields inside, and is able to trigger validation on those fields when the Collapse is closed to show an error.
  • Easily disables single or grouping form fields in the root component of the page by specifying their paths.
